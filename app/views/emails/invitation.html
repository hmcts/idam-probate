Dear {{ data['labParticipantName'] }},

{{ data['leadLabApplicantName'] }} has told us that you're an executor who wants to make a probate application to deal with the estate of {{ data['deceasedFirstName'] }} {{ data['deceasedLastName'] }}.

What you need to do

Use the link at the end of this email to view a legal declaration based on the information {{ data['leadLabApplicantName'] }} has given. You'll then need to agree that it's correct.

When you visit the link you'll be asked to enter a security code which will be sent to your mobile by SMS. If you don't receive the code within 1 hour, {{ data['leadLabApplicantName'] }} may not have given us the right mobile number.

If you don't agree with the legal statement

You should contact {{ data['leadLabApplicantName'] }} if you have any questions about the declaration, or if you don't agree that it's correct.

Only {{ data['leadLabApplicantName'] }} can make changes to the information on the legal declaration. They'll need to tell you if they've updated the declaration, and you'll be able to check it again using the link at the end of this email.

The probate registry is not able to help with any disagreements between executors. If you're not all able to agree the legal statement, you'll need to find another way to reach an agreement. This may include getting legal advice.

If you dont want to apply for probate

If you don't want to make this application, you can choose:

  • not to apply for probate, but reserve the right to apply later if another executor becomes unable to deal with the estate (this is known as having 'power reserved')
  • to completely give up your right to apply for probate (this is known as 'renunciation')
  • If you don't want to apply, you'll need to contact and ask them to change the information they've given.

Read and confirm the legal declaration
The legal declaration can be viewed at:

https://www.probate.reform.hmcts.net/invitation/williams-afr-456

You can use the link for the next 6 months.

Do not delete this email as you'll need to use the link again if the legal statement gets updated.
